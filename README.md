# <span style="color: #8a2be2; text-shadow: 2px 2px 4px rgba(0, 0, 0, 0.3);">**Explore-With-Me**</span>
Микросервисное приложение, которое позволяет пользователям делиться событиями и находить компанию для участия в мероприятиях.

-----------------------------------------------------------------------------------------------------------------------------

## Feature "Comments"

### <a href="https://github.com/Elena-Zhorzhevna/java-explore-with-me/pull/3">Pull Request Link</a>

Приложение позволяет пользователям делиться информацией об интересных событиях и находить компанию для участия в них.

Функциональность комментариев позволяет пользователям оставлять отзывы, делиться мнениями или задавать вопросы по 
поводу различных событий.

В базе данных имеется таблица comments, которая хранит комментарии. Каждый комментарий привязан к определенному 
событию и пользователю, который его оставил.

Доступ к функциональности комментариев осуществляется через несколько эндпоинтов, разделённых на категории Private, 
Admin и Public API, что позволяет управлять правами доступа в зависимости от роли пользователя.
